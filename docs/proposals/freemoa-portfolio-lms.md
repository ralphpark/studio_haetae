교육 상담 통합 관리 플랫폼 (LMS)

대학교 취업 상담 프로그램을 위한 풀스택 웹 플랫폼입니다. 학생-상담사-코디네이터-관리자 4개 역할 기반으로 설계되어, 상담 예약부터 화상 상담, 만족도 조사, 통계 분석까지 교육 상담의 전 과정을 하나의 시스템에서 관리합니다.

[주요 기능]

1. 학생 포털
- 상담사 프로필 조회 및 캘린더 기반 상담 예약
- 온라인 강의(코스) 수강 및 진도 관리
- Zoom 기반 화상 상담 참여
- 상담 가이드 및 공지사항 열람

2. 상담사 포털
- 월별 상담 일정 관리 및 가용 시간 설정
- 상담 건별 기록 작성 및 이력 관리
- Zoom 링크 자동 생성 및 화상 상담 운영
- 잔여 상담 횟수 실시간 확인

3. 코디네이터 포털
- 전체 상담 현황 모니터링 및 배정 관리
- 상담사별/유형별 통계 분석
- 만족도 조사 관리 및 결과 분석
- 학생/상담사 계정 관리

4. 관리자 대시보드
- 상담 유형별 통계 시각화 (월별/유형별 차트)
- 상담사 계정 및 프로필 관리
- 공지사항/FAQ/가이드 콘텐츠 관리(WYSIWYG 에디터)
- 만족도 조사 설계 및 통계 분석
- 월별 상담 한도 설정, 대기열 관리
- 파일 스토리지 관리 (AWS S3)
- 사이트 설정 및 권한 관리

[기술 스택]
- Frontend: React, TypeScript, Tailwind CSS, shadcn/ui, FullCalendar
- Backend: Express.js, Drizzle ORM, PostgreSQL (Neon)
- 인프라: AWS S3, Zoom API, Google Calendar 연동
- 배포: Vercel

[프로젝트 특징]
- 4개 역할(학생/상담사/코디네이터/관리자) 기반 권한 분리
- 캘린더 기반 실시간 예약 시스템
- Zoom API 연동 화상 상담
- 만족도 조사 자동화 및 통계 시각화
- WYSIWYG 에디터(TinyMCE) 기반 콘텐츠 관리
- 반응형 웹 (모바일/태블릿/PC)
