**제안사**: Studio HaeTae (https://haetae.studio)

## 1. 프로젝트 이해

### 사업 배경

교회 내부 운영 효율화를 위해 **성도 관리(교적)와 재정 관리** 기능을 통합한 웹 기반 관리 시스템을 신규 구축합니다. 기존 운영 중인 웹뷰 기반 모바일 앱과의 API 연동을 통해 모바일에서도 주요 기능을 활용할 수 있도록 합니다.

### 핵심 가치

- 교적(성도) 관리와 재정 관리를 하나의 시스템으로 통합
- 권한별 접근 제어로 관리자/교역자/재정부 역할 분리
- 기존 웹뷰 앱과 API 연동으로 모바일 활용성 확보

### 타겟 사용자

| 구분 | 대상 | 주요 기능 |
|------|------|----------|
| 관리자 | 담임 목사/사무장 | 전체 대시보드, 통계, 시스템 설정 |
| 교역자 | 부교역자/전도사 | 교적 관리, 심방 기록, 출석 |
| 재정부 | 재정 담당 직분자 | 헌금/지출 관리, 결산 보고서, 기부금 영수증 |

### 현재 준비 상태

- 기획 및 디자인(Figma) 완료 → 개발 중심 진행 가능
- 내부 IT 담당 인력 보유 → 기술 협의 가능
- 기존 웹뷰(하이브리드) 앱 운영 중

## 2. 단계별 개발 제안

Figma 디자인 제공 기준, **3단계에 걸쳐 10주 내** 구축합니다.

---

### 1단계: 교적 관리 시스템

> 성도 정보, 교회 조직, 심방 기록 - 교회 행정의 핵심

| 기능 | 상세 내용 | 공수(MD) | 금액 |
|------|----------|---------|------|
| 성도 정보 관리 | 인적 정보 등록/수정/삭제, 가족 관계 연결, 사진 관리 | 4 | 200만원 |
| 교회 조직 관리 | 교구/구역/직분 등 조직 구조 관리, 직분 이력 | 3 | 150만원 |
| 심방/활동 기록 | 심방 기록 등록, 성도별 활동 히스토리, 타임라인 뷰 | 2 | 100만원 |
| 검색 + 통계 | 다양한 필터 기반 성도 검색, 성도 통계 (연령/지역/직분 분포) | 2 | 100만원 |
| 반응형 퍼블리싱 | Figma 디자인 기반 퍼블리싱 | 2 | 100만원 |
| **소계** | | **13MD** | **650만원** |

**주요 특징**
- 교구/구역/직분 기반 계층적 조직 관리
- 심방 기록을 성도별 타임라인으로 조회
- 다중 조건 필터 검색 (연령, 직분, 지역, 등록일 등)
- **납기: 계약일로부터 4주**

### 2단계: 재정 관리 시스템

> 헌금, 지출, 결산, 기부금 영수증 - 교회 재정의 투명한 관리

| 기능 | 상세 내용 | 공수(MD) | 금액 |
|------|----------|---------|------|
| 헌금 수입 관리 | 헌금 종류별(십일조/감사/건축 등) 수입 등록 및 관리, 성도별 헌금 내역 | 3 | 150만원 |
| 지출 관리 | 지출 결의서 작성, 항목별 지출 관리, 결재 프로세스 | 3 | 150만원 |
| 결산 보고서 | 예산 대비 결산 보고서 자동 생성, 월별/연별 재정 현황 | 3 | 150만원 |
| 기부금 영수증 | 연말정산용 기부금 영수증 발급 기능, PDF 출력 | 3 | 150만원 |
| **소계** | | **12MD** | **600만원** |

**주요 특징**
- 헌금 종류별 분류 및 성도별 헌금 이력 추적
- 지출 결의서 → 승인 → 집행 프로세스
- 예산 대비 결산 보고서 자동 생성 (PDF)
- 연말정산 기부금 영수증 일괄/개별 발급
- **납기: 1단계 완료 후 3주**

### 3단계: 관리자 대시보드 + 앱 연동

| 기능 | 상세 내용 | 공수(MD) | 금액 |
|------|----------|---------|------|

| 기존 앱 API 연동 | 웹뷰 앱과 신규 시스템 간 REST API 연동, 데이터 동기화 | 3 | 150만원 |

- 성도 현황 + 재정 현황을 한눈에 보는 통합 대시보드
