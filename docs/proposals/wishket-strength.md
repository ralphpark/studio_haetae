# 위시켓 전문 분야 및 차별화된 강점 (5000자)

## 전문 분야

1. 풀스택 웹 개발 (프론트엔드 + 백엔드 + 배포)

Next.js와 React를 중심으로 프론트엔드부터 백엔드, 데이터베이스, 배포까지 전 영역을 한 사람이 담당합니다. 기획 의도를 가장 잘 아는 사람이 직접 코드를 작성하기 때문에, 에이전시처럼 여러 사람을 거치면서 발생하는 의도 왜곡이 없습니다.

프론트엔드에서는 Next.js의 SSR/SSG/ISR을 상황에 맞게 활용하여 성능과 SEO를 동시에 잡습니다. Tailwind CSS로 빠르게 반응형 UI를 구축하고, Framer Motion과 Three.js로 인터랙티브한 사용자 경험을 만듭니다.

백엔드에서는 Supabase(PostgreSQL)를 메인 데이터베이스로, Edge Functions와 REST API로 서버 로직을 처리합니다. Drizzle ORM을 활용한 타입 안전한 쿼리, Row Level Security(RLS) 기반 보안 정책, 실시간 구독(Realtime) 등 Supabase의 기능을 깊이 있게 활용합니다.

배포는 Vercel을 기본으로 사용합니다. GitHub 연동 자동 배포, CDN, SSL 인증서, 프리뷰 배포까지 포함되어 있어 별도 서버 관리 비용 없이 안정적으로 운영할 수 있습니다.


2. 이커머스 및 쇼핑몰

단순 상품 나열이 아닌, 브랜드의 세계관을 디지털로 옮기는 커머스를 만듭니다. 실제로 럭셔리 화장품 D2C 커머스에서 Three.js 3D 제품 뷰어와 몰입형 캐러셀을 구현한 경험이 있습니다.

결제 시스템은 토스페이먼츠, 나이스페이, PortOne 등 국내 주요 PG사 연동 경험이 있습니다. 카드 결제, 간편결제(카카오페이/네이버페이), 가상계좌까지 대응 가능합니다. 주문/배송 관리, 상품 리뷰/평점, 쿠폰/할인, 회원 등급별 차등 가격 등 쇼핑몰 운영에 필요한 전체 기능을 구현합니다.

카페24나 아임웹 같은 솔루션 기반 빠른 셋팅도 가능합니다. 예산이 제한적이거나 빠른 오픈이 필요한 경우 3~5일 내에 스킨 커스터마이징부터 상품 등록, PG 결제, 배송 설정까지 완료합니다.


3. SaaS 및 비즈니스 플랫폼

단순 홈페이지를 넘어서, 복잡한 비즈니스 로직이 포함된 웹 플랫폼을 설계하고 개발합니다. 실제로 구축한 플랫폼 유형은 다음과 같습니다.

교육 LMS: 4개 역할(학생/상담사/코디네이터/관리자) 기반 상담 예약, Zoom 화상 상담, 만족도 조사까지 통합한 올인원 교육 포털. PostgreSQL 트랜잭션으로 예약 동시성(오버부킹) 문제를 해결했습니다.

퍼포먼스 마케팅 SaaS: YouTube/Instagram/TikTok API 연동 실시간 조회수 검증, PortOne 파트너 정산 시스템 자동화, Gemini AI 콘텐츠 분석. 크리에이터와 대행사 간 투명한 성과 측정 플랫폼입니다.

로컬 커머스 O2O: Gemini AI 챗봇 추천, Google Maps 기반 주변 매장 입찰, 이중 결제 연동, 판매자 온보딩 프로세스까지 풀스택으로 완성했습니다.

이러한 경험을 바탕으로 인력 매칭, 예약/스케줄링, 커뮤니티, 검사/설문 시스템 등 다양한 형태의 플랫폼을 빠르게 구축할 수 있습니다.


4. AI/LLM 연동 및 자동화

AI를 단순히 도구로 사용하는 것이 아니라, 프로덕트의 핵심 기능으로 통합합니다. OpenAI GPT, Anthropic Claude, Google Gemini 등 주요 LLM API를 모두 다룰 수 있으며, 9개 AI 모델을 실시간으로 연동하는 트레이딩 플랫폼을 직접 구축하여 운영 중입니다.

RAG(검색 증강 생성) 파이프라인을 구축하여 자체 데이터 기반의 정확한 AI 답변 시스템을 만듭니다. Supabase pgvector로 벡터 DB를 구성하고, LangChain으로 검색과 생성을 연결합니다. 할루시네이션을 최소화하고 출처를 표시하여 신뢰도를 확보합니다.

n8n 워크플로우 자동화도 전문 분야입니다. 현재 직접 n8n 서버를 셀프 호스팅하여 다음 시스템들을 24시간 운영 중입니다.

인스타그램/쓰레드 자동 포스팅: Claude가 콘텐츠를 생성하고, Gemini가 이미지를 생성하고, Puppeteer가 한글 카드뉴스를 렌더링하여 하루 5회 자동 발행합니다.

AI 트레이딩 봇: 15분 주기로 시장 데이터를 수집하고, 9개 AI 모델이 의사결정을 내리고, 자동으로 거래를 실행합니다.

뉴스 자동 수집: RSS 피드에서 최신 IT 뉴스를 수집하고, Claude가 가장 관련성 높은 뉴스를 선정하여 DB에 자동 입력합니다.

이처럼 직접 운영하는 시스템이 있기 때문에, 자동화 프로젝트의 실제 운영 이슈(장애 대응, 토큰 만료, API 제한, 스케줄 관리)를 잘 알고 있습니다.


## 차별화된 강점

1. 연구원의 체계성 + CTO의 실행력

대기업 연구원 12년의 경험은 체계적인 문서화, 엄격한 품질 관리, 대규모 데이터 처리 역량으로 이어집니다. 요구사항 정의서, 기능 명세서, 테스트 시나리오를 꼼꼼하게 작성합니다.

스타트업 CTO의 경험은 빠른 의사결정, MVP 우선 전략, 린 개발 방법론으로 이어집니다. 완벽한 제품을 오래 만드는 것보다, 핵심 기능을 빠르게 만들어 시장에서 검증하는 것이 더 중요하다는 것을 실전에서 배웠습니다.

이 두 가지의 조합이 Studio HaeTae의 가장 큰 강점입니다. 체계적이면서도 빠르게, 꼼꼼하면서도 유연하게 프로젝트를 진행합니다.


2. 납품 후에도 돌아가는 코드

많은 외주 프로젝트가 납품 후 문제가 됩니다. 작동은 하지만 유지보수가 불가능한 코드, 서버 비용이 계속 올라가는 구조, 문서 없이 인수인계되는 상황.

Studio HaeTae는 다릅니다. TypeScript로 타입 안전성을 확보하고, 컴포넌트 구조를 깔끔하게 유지하며, 주석과 README를 충실히 작성합니다. Vercel 배포로 서버 관리 부담을 없애고, Supabase로 DB 관리를 단순화합니다.

소스 코드 100%를 인수인계하고, 관리자 페이지 사용법을 교육하며, 1개월 무상 버그 수정을 제공합니다. 납품이 끝이 아니라, 안정적인 운영의 시작이 되도록 합니다.


3. 단계별 개발로 리스크 최소화

처음부터 전체 기능을 한 번에 만들지 않습니다. 1단계(MVP)에서 핵심 기능만 빠르게 구현하여 검증하고, 검증된 후에 2단계, 3단계로 확장합니다.

이 방식의 장점은 명확합니다. 초기 투자 비용을 줄이고, 시장 반응을 보면서 방향을 수정할 수 있으며, 불필요한 기능에 예산을 낭비하지 않습니다. 각 단계는 독립적으로 계약 가능하여, 1단계 결과를 보고 계속 진행할지 결정하실 수 있습니다.


4. 다양한 산업 도메인 경험

교육(LMS, 상담 시스템), 이커머스(D2C 커머스, 로컬 O2O), 마케팅(퍼포먼스 SaaS, SNS 자동화), 핀테크(AI 트레이딩), 의료(의료기기 홈페이지), 공연(PNL 분석 시스템), 종교(교회 관리 시스템) 등 다양한 산업 도메인에서 프로젝트를 수행했습니다.

각 산업의 특성과 요구사항을 이해하고 있기 때문에, 도메인 학습에 드는 시간이 짧고 핵심을 빠르게 파악합니다.


5. 투명한 커뮤니케이션

프로젝트 진행 중 어떤 단계에 있는지, 어떤 이슈가 있는지를 숨기지 않습니다. 주 1~2회 시연을 통해 진행 상황을 시각적으로 공유하고, 이슈가 발생하면 즉시 알려드립니다.

견적도 투명합니다. 기능별 MD(공수) 기반으로 산정하여, 어떤 기능에 얼마의 비용이 드는지 명확하게 안내합니다. 숨겨진 추가 비용 없이, 합의된 범위 안에서 작업합니다.


6. 최신 기술 스택에 대한 깊은 이해

단순히 프레임워크를 사용하는 것이 아니라, 왜 이 기술을 선택하는지 설명할 수 있습니다.

Next.js를 선택하는 이유: SSR로 SEO를 확보하면서, SSG/ISR로 성능을 최적화할 수 있습니다. App Router 기반의 서버 컴포넌트로 초기 로딩 속도를 극적으로 개선합니다. API Routes로 별도 백엔드 서버 없이 풀스택 개발이 가능합니다.

Supabase를 선택하는 이유: PostgreSQL의 안정성 위에 인증, 스토리지, 실시간 구독, Edge Functions를 모두 제공합니다. Firebase 대비 오픈소스이고, 데이터 소유권이 클라이언트에게 있습니다. RLS(Row Level Security)로 보안을 데이터베이스 레벨에서 처리합니다.

Vercel을 선택하는 이유: GitHub 푸시만으로 자동 배포됩니다. 글로벌 CDN, SSL, 프리뷰 배포가 기본 포함되어 별도 서버 관리가 불필요합니다. 소규모 프로젝트는 무료로 운영 가능합니다.

이러한 기술 선택의 근거를 의뢰인에게 투명하게 설명드리고, 프로젝트 특성에 맞는 최적의 기술 조합을 제안합니다.


7. 자동화 마인드셋

개발 업무 자체도 자동화합니다. 숨고/크몽/프리모아에서 의뢰가 오면, 자체 구축한 스킬 시스템이 NotebookLM에서 단가를 조회하고, 노션에 제안서를 자동 생성하고, 견적 메시지까지 만들어줍니다. 인스타그램 마케팅도 AI가 콘텐츠를 생성하고 자동 발행합니다.

이런 자동화 마인드셋은 클라이언트 프로젝트에도 그대로 적용됩니다. 반복적인 업무는 시스템으로 해결하고, 사람은 창의적인 일에 집중할 수 있도록 설계합니다. 단순히 기능을 만드는 것이 아니라, 운영 효율을 높이는 시스템을 만듭니다.
