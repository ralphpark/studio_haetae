## 1. 프로젝트 이해

### 사업 배경

글로벌 K-POP 팬덤을 대상으로 **아이돌/배우 굿즈 온라인 뽑기(라플/가챠)** 웹 서비스를 구축합니다. 꽝 없는 100% 당첨 방식으로, 확률 조작 방지 및 투명성 증빙 로그 시스템이 핵심입니다.

### 핵심 가치

- K-POP 굿즈 특화 UI/UX + 실시간 당첨 이벤트
- 조작 방지 보안 로그로 신뢰도 확보
- 글로벌 서비스 (한국어/일본어/영어 3개국어)

### 타겟 사용자

| 구분 | 대상 | 주요 행동 |
|------|------|----------|
| 메인 유저 | 국내외 K-POP 팬덤 (10~30대) | 포인트 충전 → 굿즈 뽑기 → 당첨 확인 |
| 관리자 | 서비스 운영자 | 상품/확률 설정, 매출 통계, 로그 모니터링 |

## 2. 단계별 개발 제안

클라이언트 디자인 제공 기준, **3단계에 걸쳐 4주 내** 구축합니다.

### 1단계: 핵심 뽑기 시스템

> 로그인, 뽑기 로직, 결제/포인트 - 서비스의 핵심 MVP

| 기능 | 상세 내용 | 공수(MD) | 금액 |
|------|----------|---------|------|
| 회원 시스템 | 회원가입/로그인, 소셜 로그인(카카오/구글), 마이페이지 | 2 | 100만원 |
| 뽑기 핵심 로직 | 확률 기반 랜덤 뽑기, 100% 당첨(꽝 없음), 조작 방지 보안 로그, 투명성 증빙 시스템 | 3 | 150만원 |
| 결제 + 포인트 | 국내 PG사 연동(신용카드, 간편결제), 포인트 충전(선불금), 결제 내역 관리 | 3 | 150만원 |
| 반응형 퍼블리싱 | 클라이언트 제공 디자인 기반 퍼블리싱 (모바일 우선) | 1 | 50만원 |
| **소계** | | **9MD** | **450만원** |

**주요 특징**
- 서버 사이드 랜덤 로직으로 클라이언트 조작 원천 차단
- 모든 뽑기 결과를 해시값과 함께 로그 기록 (투명성 증빙)
- 10대 메인 타겟에 맞춘 모바일 우선 반응형
- **납기: 계약일로부터 2주**
