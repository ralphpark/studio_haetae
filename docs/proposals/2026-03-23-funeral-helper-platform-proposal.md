# 목포권 장례도우미 인력파견 플랫폼 제안서

**제안사**: Studio HaeTae (https://haetae.studio)
**제안일**: 2026년 3월 23일
**유효기간**: 제안일로부터 30일

---

## 1. 프로젝트 이해

### 사업 배경
목포시를 중심으로 강진군, 무안군, 함평군, 영암군을 커버하는 **장례도우미 인력파견 사업**을 위한 전문 웹 플랫폼을 구축합니다.

### 핵심 가치
- 전남 서남권 유일의 장례도우미 전문 매칭 서비스
- 24시간 긴급 파견 대응 체계
- 지역 밀착형 신뢰 기반 서비스

### 타겟 사용자
| 구분 | 대상 | 니즈 |
|------|------|------|
| **B2B** | 장례식장 운영자 | 안정적 도우미 인력 확보 |
| **B2C** | 유가족 | 급한 상황에서 빠른 도우미 요청 |
| **공급** | 장례도우미 구직자 | 안정적 일감 확보, 일정 관리 |

## 2. 단계별 개발 제안

MVP(최소 기능 제품) 방식으로 **3단계에 걸쳐** 점진적으로 구축합니다.
각 단계는 독립적으로 운영 가능하며, 사업 성장에 맞춰 확장할 수 있습니다.

---

### 1단계: 랜딩페이지 + 문의 시스템

> 사업 홍보와 고객 유입을 위한 기본 온라인 거점

| 기능 | 상세 내용 | 공수(MD) | 금액 |
|------|----------|---------|------|
| 메인 랜딩페이지 | 히어로 섹션, 서비스 소개, 커버 지역 지도(목포/강진/무안/함평/영암), FAQ | 3 | 150만원 |
| 문의 시스템 | 웹 문의폼 + 카카오톡 채널 연동 + 전화 바로걸기(모바일) | 1 | 50만원 |
| 반응형 + SEO | 모바일 우선 반응형 디자인, 지역 키워드 검색엔진 최적화 | 1 | 50만원 |
| **소계** | | **5MD** | **250만원** |

**주요 특징**:
- 모바일 우선 설계 (유가족은 급한 상황에서 스마트폰으로 검색)
- 지역 SEO 최적화 ("목포 장례도우미", "무안 장례 인력파견" 등)
- 절제되고 신뢰감 있는 디자인 톤

**납기**: 계약일로부터 2주

## 6. 진행 프로세스

```
계약 체결 → 요구사항 정의 → 디자인 시안 → 개발 → 중간 검수 → 수정/보완 → 최종 납품 → 무상 유지보수
```

1. **요구사항 정의** (1~2일): 상세 기능 협의 및 요구사항 정의서 작성
2. **디자인 시안** (2~3일): 주요 화면 디자인 시안 제공 및 피드백 반영
3. **개발**: 단계별 납기 기준 진행
4. **중간 검수**: 개발 50% 시점에 중간 시연 및 피드백 수렴
5. **최종 납품**: 테스트 완료 후 실서버 배포 및 인수인계
