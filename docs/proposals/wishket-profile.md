# 위시켓 회사소개 (5000자)

## Studio HaeTae - 기획부터 배포까지, 원스톱 풀스택 개발 스튜디오

안녕하세요, Studio HaeTae 대표 개발자 박근수입니다.

12년간 대기업 연구원으로 근무하며 대규모 시스템의 설계와 운영을 경험했고, 현재는 스타트업 CTO로서 실전 프로덕트를 직접 기획하고 개발하고 있습니다. 연구원 시절 쌓은 체계적인 문제 해결 능력과, 스타트업 CTO로서의 빠른 실행력을 겸비하고 있습니다.

Studio HaeTae는 이러한 경험을 바탕으로, 기획부터 디자인, 프론트엔드, 백엔드, 배포, 운영까지 한 사람이 직접 수행하는 원스톱 풀스택 개발 스튜디오입니다.


[왜 Studio HaeTae인가]

1. 12년 대기업 연구원 + 현직 스타트업 CTO

대기업 연구원으로 12년간 근무하면서 대규모 프로젝트의 설계, 개발, 운영 전 과정을 경험했습니다. 수천만 건의 데이터를 다루는 시스템을 설계하고, 엄격한 코드 리뷰와 품질 관리 프로세스를 체득했습니다. 다양한 산업 도메인(제조, 물류, 유통, 교육)의 프로젝트에 참여하며 기술을 비즈니스에 연결하는 역량을 키웠습니다.

현재는 스타트업 CTO로 근무하며, 제한된 자원 안에서 빠르게 MVP를 만들고 시장에서 검증하는 실전 경험을 쌓고 있습니다. 기술 스택 선정부터 아키텍처 설계, 팀 빌딩, 배포 전략까지 프로덕트의 전 생애주기를 직접 관리하고 있습니다. 대기업의 안정성과 스타트업의 속도, 두 가지를 모두 갖춘 개발자입니다.


2. 중간 과정 없는 직접 커뮤니케이션

에이전시에 의뢰하면 영업 담당자, PM, 디자이너, 개발자로 이어지는 긴 커뮤니케이션 체인이 생깁니다. 전달 과정에서 의도가 왜곡되고, 시간과 비용이 낭비됩니다. 한 번의 수정 요청이 여러 단계를 거치면서 일주일이 걸리는 경우도 흔합니다.

Studio HaeTae는 대표 개발자가 직접 상담하고, 직접 기획하고, 직접 개발합니다. 의뢰인의 의도를 100% 이해한 사람이 코드를 작성하기 때문에, 결과물의 완성도가 높고 수정이 최소화됩니다. 카카오톡이나 메일로 요청하시면 당일 또는 익일 내에 반영이 가능합니다.


3. 비즈니스를 이해하는 개발자

단순히 시키는 대로 코딩하는 개발자가 아닙니다. 12년간 다양한 산업 도메인의 프로젝트를 경험하면서, 기술을 비즈니스 관점에서 바라보는 시각을 갖추었습니다.

이 기능이 정말 필요한가, 사용자가 실제로 이렇게 쓸까, MVP로 먼저 검증하고 확장하는 게 낫지 않을까. 이런 질문을 먼저 던지는 개발자입니다. 불필요한 기능을 줄이고 핵심에 집중하여 예산과 시간을 절약해 드립니다. 단계별 개발 방식으로 초기 부담을 최소화하고, 검증된 후에 확장하는 전략을 제안합니다.


4. 직접 운영하는 시스템이 있는 개발자

남의 프로젝트만 만들어주는 것이 아니라, 직접 서비스를 기획하고 운영하고 있습니다. AI 트레이딩 플랫폼, 인스타그램 자동 포스팅 시스템, 뉴스 자동 수집 시스템 등을 직접 구축하여 24시간 운영 중입니다. 실제 운영에서 발생하는 장애 대응, 성능 최적화, 보안 이슈를 직접 겪고 해결한 경험이 있기 때문에, 납품 후에도 안정적으로 운영될 수 있는 코드를 작성합니다.
