## 클라이언트 질문 답변

### Q1. 유사한 프로젝트를 수행한 경험이 있다면 무엇입니까? (500자)

자체 에이전시(Studio HaeTae) 운영 중 n8n 기반 업무 자동화 시스템을 직접 설계·구축하여 운영하고 있습니다. 클라이언트 문의 접수 → CRM 등록 → 견적서 자동 생성 → 이메일 발송 → 리마인더까지의 전체 파이프라인을 n8n 워크플로우로 자동화했습니다. 또한 콘텐츠 마케팅 자동화 시스템도 구축하여, RSS 뉴스 수집 → AI(Claude) 콘텐츠 생성 → 이미지 렌더링 → SNS 자동 발행 → 텔레그램 알림까지 하루 3회 완전 자동으로 운영 중입니다. Notion API, Supabase, Gmail, 텔레그램 등 다양한 외부 서비스를 n8n으로 연동한 실무 경험이 있으며, 직원 교육용 매뉴얼 작성 및 온보딩 지원 경험도 보유하고 있습니다.

### Q2. 유사한 기술을 사용한 경험에 대해 설명해주세요. (500자)

n8n(자체 호스팅 자동화 플랫폼)을 2년간 실무에서 활용하고 있습니다. Zapier와 동일한 기능을 제공하면서도 자체 호스팅이 가능해 월 구독료 없이 무제한 워크플로우를 운영할 수 있습니다. 현재 운영 중인 워크플로우: (1) 뉴스 RSS 수집 → Claude AI 분석 → 콘텐츠 자동 생성, (2) 고객 문의 Webhook → Notion DB 등록 → 자동 회신 이메일, (3) 견적서 데이터 입력 → PDF 자동 생성 → 이메일 발송, (4) 일정 기반 리마인더 자동 발송. HubSpot 대안으로 Notion CRM 또는 Supabase 기반 경량 CRM 구축 경험이 있으며, 프로젝트 성격에 따라 HubSpot Free + n8n 조합도 가능합니다. 클라이언트의 기존 엑셀 데이터를 CRM으로 마이그레이션한 경험도 다수 보유하고 있습니다.

### Q3. 이 프로젝트를 성공적으로 수행할 수 있는 제안이 있으십니까? (500자)

Zapier 대신 n8n을 제안드립니다. Zapier는 워크플로우당 월 과금(Pro 기준 월 $29~)이 발생하지만, n8n은 자체 호스팅 시 무료이며 동일 기능을 제공합니다. 해외 화장품 인증 대행이라는 업종 특성상 고객별 인증 진행 상태 관리, 서류 제출 기한 리마인더, 견적·인보이스 자동 발행이 핵심인데, n8n의 조건 분기(IF/Switch) + 스케줄러(Cron) + 이메일 노드 조합으로 모두 구현 가능합니다. CRM은 HubSpot Free 또는 Notion을 활용하고, n8n이 이를 연결하는 자동화 허브 역할을 합니다. 추후 워크플로우 추가나 수정도 코딩 없이 가능하여, 교육 후 내부 직원이 직접 관리할 수 있습니다. 납품 시 전체 워크플로우 설계서 + 직원 매뉴얼 + 교육 세션을 포함합니다.

**1. Studio HaeTae 업무 자동화 시스템 (n8n)**
자체 에이전시 운영을 위해 n8n 기반 업무 자동화 파이프라인을 직접 설계·구축했습니다. 고객 문의 Webhook 수신 → Notion CRM 자동 등록 → 견적서 AI 생성 → PDF 변환 → 이메일 자동 발송까지 전 과정을 자동화했으며, Supabase DB + n8n + Claude AI + Gmail을 연동한 실무 운영 중입니다.

**2. SNS 콘텐츠 마케팅 자동화 (n8n + Claude AI)**
RSS 뉴스 수집 → AI 콘텐츠 생성 → 이미지 렌더링 → Instagram/Threads 자동 발행 → 텔레그램 리포트까지의 완전 자동화 파이프라인을 구축하여 하루 3회 무인 운영 중입니다. 스케줄링, 조건 분기, 에러 핸들링, 외부 API 연동 등 본 프로젝트에 필요한 모든 자동화 패턴을 실무에서 검증했습니다.

**3. 퍼포먼스 마케팅 SaaS (외부 API 연동 + 자동화 대시보드)**
YouTube·Instagram·TikTok API를 연동하여 데이터를 자동 수집하고, AI 분석 후 대시보드에 시각화하는 B2B SaaS입니다. 외부 서비스 연동 → 자동 데이터 처리 → 리포팅이라는 본 프로젝트의 핵심 구조와 동일합니다.

## 사용 기술과 툴

**CRM**: HubSpot Free CRM
→ 고객 DB 관리, 파이프라인(인증 진행 단계) 관리, 거래 추적에 최적. 무료 플랜으로 충분하며, 추후 Marketing Hub 등으로 확장 가능.

**업무 자동화**: n8n (자체 호스팅)
→ Zapier와 동일한 노코드 자동화 플랫폼이나, 자체 호스팅으로 월 구독료 없이 무제한 워크플로우 운영. 400개+ 서비스 연동 지원. HubSpot, Gmail, Notion, Google Sheets, Slack 등 본 프로젝트에 필요한 모든 서비스와 네이티브 연동.

**문서 자동화**: Google Docs/Sheets + n8n
→ 견적서·인보이스 템플릿을 Google Docs로 관리하고, n8n이 고객 데이터를 자동 삽입하여 PDF 생성 및 이메일 발송.

**커뮤니케이션 자동화**: Gmail + n8n Cron 스케줄러
→ 인증 진행 단계별 리마인더, 서류 제출 기한 알림, 완료 안내 등을 조건 기반으로 자동 발송.

**인프라**: 클라우드 서버 (월 1~2만원)
→ n8n 자체 호스팅용. 안정적 24/7 운영.

**커뮤니케이션**: Slack 또는 카카오톡 + Notion (진행상황 공유)

## 관련 포트폴리오 설명 (2000자)

**1. Studio HaeTae 업무 자동화 시스템 (n8n 기반 CRM + 자동화)**

자체 IT 에이전시를 운영하면서 수작업으로 처리하던 고객 관리, 견적서 발행, 후속 연락 업무를 n8n 기반으로 완전 자동화한 프로젝트입니다. 본 프로젝트와 업무 성격이 거의 동일합니다.

도입 전에는 고객 문의가 들어오면 수동으로 스프레드시트에 기록하고, 견적서를 직접 작성하여 이메일로 보내고, 후속 리마인더도 캘린더에 메모해두는 방식이었습니다. 업무 누락이 빈번했고, 특히 견적 발송 후 후속 연락 타이밍을 놓치는 경우가 많았습니다.

n8n으로 구축한 자동화 워크플로우는 다음과 같습니다. (1) 웹사이트 상담 폼 제출 시 Webhook으로 자동 수신 → (2) Notion CRM에 고객 정보 자동 등록 및 상태를 '신규 문의'로 설정 → (3) Claude AI가 문의 내용을 분석하여 견적서 초안 자동 생성 → (4) PDF로 변환 후 Gmail을 통해 고객에게 자동 발송 → (5) 3일 후 미응답 시 리마인더 자동 발송 → (6) 텔레그램으로 담당자에게 현황 리포트 발송.

이 시스템 도입 후 고객 응대 시간이 평균 2시간에서 15분으로 단축되었고, 후속 연락 누락률이 0%로 개선되었습니다. 본 프로젝트의 '견적/인보이스 자동 발행', '리마인더 자동 발송', '프로세스 표준화' 요건을 이미 실무에서 구현하고 운영한 경험입니다.

**2. SNS 콘텐츠 마케팅 완전 자동화 파이프라인 (n8n + AI)**

n8n을 중심으로 6개 외부 서비스(RSS, Claude AI, Gemini, Supabase, Instagram API, Telegram)를 연동하여, 콘텐츠 기획부터 발행까지 전 과정을 자동화한 시스템입니다.

매일 오전 7시에 n8n 스케줄러가 RSS 피드(GeekNews, HackerNews, Dev.to 등)에서 뉴스를 수집하고, Claude AI가 에이전시 톤에 맞는 콘텐츠 2건을 자동 선정합니다. 선정된 주제로 Instagram용 카드뉴스 카피와 Threads용 텍스트 포스트를 AI가 생성하고, Puppeteer가 HTML 템플릿을 이미지로 렌더링합니다. 완성된 콘텐츠는 Supabase에 저장되어 하루 3회(07:30, 12:30, 19:00) 자동 발행되며, 발행 결과는 텔레그램으로 리포트됩니다.

이 프로젝트에서 검증한 핵심 패턴은 (1) Cron 기반 정기 자동 실행, (2) 조건 분기를 통한 워크플로우 제어(요일별 다른 콘텐츠 타입), (3) 외부 API 에러 시 재시도 로직, (4) 실행 결과 모니터링 및 알림입니다. 이 패턴들은 본 프로젝트의 인증 단계별 리마인더, 조건부 견적서 발행, 실패 시 재시도 등에 그대로 적용됩니다.

**3. 퍼포먼스 마케팅 SaaS (외부 데이터 수집 + AI 분석 + B2B 대시보드)**

YouTube, Instagram, TikTok 공식 API를 주기적으로 호출하여 크리에이터 퍼포먼스 데이터를 자동 수집하고, Gemini AI로 콘텐츠를 분석한 뒤 B2B 관리자 대시보드에 시각화하는 SaaS입니다. '외부 데이터 자동 수집 → AI 처리 → 내부 시스템 반영'이라는 자동화 파이프라인의 설계·구현 역량을 증명하는 프로젝트이며, PortOne 파트너 정산 자동화(인보이스 자동 생성·정산)까지 구현하여 본 프로젝트의 견적/인보이스 자동화 요건과도 직접적으로 연결됩니다. React + Express + PostgreSQL + Drizzle ORM 기반.
