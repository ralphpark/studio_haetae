# 위시켓 회사 구성 및 직원 소개

## 회사 구성

Studio HaeTae는 대표 개발자 1인이 기획부터 배포까지 전 과정을 직접 수행하는 풀스택 개발 스튜디오입니다. 에이전시처럼 여러 명이 분업하는 구조가 아니라, 한 사람이 의뢰인의 의도를 100% 이해한 상태에서 전체를 일관되게 구현합니다. 이 구조의 장점은 커뮤니케이션 비용이 제로에 가깝고, 설계 의도가 코드까지 왜곡 없이 전달된다는 점입니다.

프로젝트 규모나 요구사항에 따라 디자인, 모바일 앱, 인프라 등 특정 영역의 검증된 전문 프리랜서와 협업하여 유연하게 팀을 구성합니다.


## 풀스택 개발자 1명

직책: 대표 개발자 / 현직 스타트업 CTO (겸직)
경력: 12년 대기업 근무 + 현직 스타트업 CTO


[경력 요약]

12년간 대기업에서 근무하며 대규모 프로젝트의 설계, 개발, 운영 전 과정을 경험했습니다. 수천만 건의 데이터를 처리하는 시스템을 설계하고, 제조, 물류, 유통, 교육 등 다양한 산업 도메인의 프로젝트에 참여했습니다.

현재는 스타트업 CTO로서 기술 스택 선정, 아키텍처 설계, MVP 개발, 시장 검증, 배포 전략까지 프로덕트의 전 생애주기를 직접 관리하고 있습니다. 동시에 Studio HaeTae를 운영하며 기업 홈페이지, 쇼핑몰, SaaS 플랫폼, AI 연동, 업무 자동화 등 다양한 외주 프로젝트를 수행하고 있습니다.


[기술 역량 상세]

프론트엔드 (상급)
Next.js, React, TypeScript를 메인으로 사용합니다. SSR/SSG/ISR을 상황에 맞게 활용하여 성능과 SEO를 동시에 확보합니다. Tailwind CSS로 빠르게 반응형 UI를 구축하고, Framer Motion으로 스크롤 연동 애니메이션과 페이지 전환 효과를 구현합니다. Three.js를 활용한 3D 제품 뷰어 구현 경험도 있습니다. 주요 실적으로 럭셔리 화장품 D2C 커머스에서 몰입형 3D 쇼핑 경험을 구현했습니다.

백엔드 (상급)
Node.js, Express 기반 REST API 설계를 합니다. Supabase(PostgreSQL)를 메인 데이터베이스로 사용하며, Drizzle ORM으로 타입 안전한 쿼리를 작성합니다. Row Level Security(RLS)로 데이터베이스 레벨 보안을 구현하고, Edge Functions로 서버리스 로직을 처리합니다. 교육 LMS 프로젝트에서 PostgreSQL 트랜잭션으로 예약 동시성(오버부킹) 문제를 해결한 경험이 있습니다.

결제 연동 (상급)
토스페이먼츠, 나이스페이, PortOne 등 국내 주요 PG사 연동 경험이 있습니다. 카드 결제, 간편결제(카카오페이/네이버페이), 가상계좌, 파트너 자동 정산까지 대응 가능합니다. 퍼포먼스 마케팅 SaaS에서 PortOne 기반 크리에이터 정산 시스템을 구축했습니다.

AI/LLM 연동 (상급)
OpenAI GPT, Anthropic Claude, Google Gemini 등 주요 LLM API를 모두 다룹니다. RAG(검색 증강 생성) 파이프라인 구축, pgvector 벡터 DB, LangChain 연동 경험이 있습니다. 9개 AI 모델을 실시간으로 연동하는 트레이딩 플랫폼을 직접 구축하여 운영 중입니다. 로컬 커머스에서 Gemini AI 챗봇 추천 엔진을 구현했습니다.

자동화/워크플로우 (상급)
n8n 셀프 호스팅 기반 업무 자동화 전문입니다. 현재 인스타그램/쓰레드 자동 포스팅(하루 5회), AI 트레이딩 봇(15분 주기), RSS 뉴스 자동 수집 시스템을 직접 구축하여 24시간 운영 중입니다. Claude가 콘텐츠를 생성하고, Gemini가 이미지를 생성하고, Puppeteer가 한글 카드뉴스를 렌더링하는 전체 파이프라인을 설계했습니다.

외부 API 연동 (상급)
Zoom API(화상 상담), Google Maps API(위치 기반 검색), YouTube/Instagram/TikTok API(조회수 검증), Microsoft Graph API(Outlook 일정), 카카오 알림톡, Instagram Graph API(자동 포스팅), Threads API 등 다양한 외부 서비스 연동 경험이 있습니다.

배포/인프라 (중급)
Vercel을 기본 배포 플랫폼으로 사용합니다. GitHub 연동 자동 배포, CDN, SSL, 프리뷰 배포를 활용합니다. AWS S3 파일 스토리지, Cloudflare DNS 관리, n8n 서버 셀프 호스팅 경험이 있습니다.

모바일 (중급)
React Native 기반 iOS/Android 크로스플랫폼 앱 개발이 가능합니다. 웹뷰 하이브리드 앱 구축 및 유지보수 경험이 있습니다.

디자인 (기본)
Figma를 활용한 와이어프레임 및 UI 설계가 가능합니다. 클린하고 기능적인 UI를 직접 설계하여 개발까지 이어갈 수 있습니다. 고급 브랜딩이나 비주얼 디자인이 필요한 경우 전문 디자이너와 협업합니다.


[수행 프로젝트 요약]

Commerce D2C: 럭셔리 화장품 몰입형 커머스 (Three.js 3D 뷰어)
Performance Marketing SaaS: 크리에이터 조회수 검증 및 자동 정산
Education LMS: 4역할 기반 대학교 커리어 컨설팅 플랫폼 (Zoom 연동)
Local Commerce: AI 추천 + 위치 기반 입찰 O2O 커머스 (Gemini AI + Google Maps)
Algorithmic Trading: 9개 AI 모델 실시간 트레이딩 자동화 (n8n 7개 워크플로우)
Instagram 자동화: AI 콘텐츠 생성 + 하루 5회 자동 발행 (Claude + Gemini + Puppeteer)


## 협업 파트너 네트워크

프로젝트 규모나 특수 요구사항에 따라 다음 분야의 검증된 프리랜서와 협업합니다. 협업 시에도 대표 개발자가 전체 PM을 맡아 일관된 품질과 커뮤니케이션을 유지합니다.

UI/UX 디자이너: 고급 브랜딩, 사용자 리서치, 프로토타이핑이 필요한 경우
네이티브 앱 개발자: Swift(iOS), Kotlin(Android) 순수 네이티브 개발이 필요한 경우
DevOps 엔지니어: AWS, Kubernetes 등 대규모 인프라 구축이 필요한 경우
QA 엔지니어: 대규모 프로젝트의 체계적인 테스트 시나리오 및 자동화 테스트가 필요한 경우
