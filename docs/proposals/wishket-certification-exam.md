**4단계 (8주): QA 및 납품**

## 사용 기술과 툴

**프론트엔드**: Next.js (App Router) + TypeScript + Tailwind CSS
→ SSR로 SEO 대응, 반응형 완벽 지원. IQCS+ 벤치마킹 디자인 구현에 최적.

**백엔드/DB**: Supabase (PostgreSQL + Auth + Edge Functions + Storage)
→ 회원 관리, 문제은행 DB, 시험 결과, 수료증 파일 저장을 통합 제공. RLS로 역할별 데이터 접근 제어.

**온라인 시험**: 자체 개발 (Next.js + Supabase)
→ 문제은행 랜덤 출제, 응시 타이머, 실시간 채점, 부정행위 방지(탭 이탈 감지). Supabase Realtime으로 관리자 실시간 모니터링.

**수료증 발급**: Puppeteer + HTML 템플릿 → PDF 자동 생성
→ 디자인 템플릿에 수료자 정보를 동적 삽입하여 PDF 생성. 발급 이력 DB 관리.

**결제**: PortOne V2 (구 아임포트)
→ 카드/계좌이체/가상계좌 지원. 결제 → 교육 신청 확정 → 이수 관리 자동 연동.

**배포**: Vercel (CDN + 자동 SSL)

**커뮤니케이션**: Slack 또는 카카오톡 + Notion + GitHub

## 관련 포트폴리오 설명 (2000자)

**1. 한양대학교 취업상담 포털 (LMS + 예약 시스템 + CRM 대시보드)**

본 프로젝트와 가장 직접적으로 연결되는 프로젝트입니다. 대학교 취업지원센터의 상담·교육 업무를 디지털화한 풀스택 시스템으로, 학생 900명+, 상담사, 코디네이터, 관리자 4개 역할별로 완전히 다른 대시보드를 설계·구현했습니다.

LMS 기능으로 Vimeo API를 연동하여 온라인 교육 영상 시청, 진도율 추적, 교육 이수 판정을 구현했습니다. 학생이 교육을 수료하면 시스템이 자동으로 이수 상태를 업데이트하고, 코디네이터가 관리자 대시보드에서 이수 현황을 일괄 조회할 수 있습니다. 이 구조는 본 프로젝트의 "교육 신청 → 이수 → 시험 응시 → 합격 판정 → 수료증 발급" 플로우와 동일합니다.

상담 예약 시스템에서는 상담사별 가능 시간 조회, 실시간 예약, 관리자 승인/반려, 예약 변경/취소, 48시간 전 취소 규칙 등 복잡한 비즈니스 로직을 PostgreSQL 트랜잭션 기반으로 구현하여 동시 예약 충돌을 방지했습니다. 관리자 대시보드에서는 월별 상담유형 통계, 상담사별 실적, 설문 응답 분석 등 다차원 데이터를 시각화합니다. React + Express + PostgreSQL + Drizzle ORM + WebSocket + Firebase 기반.

**2. 심다(PlantBid) - PG 결제 시스템 + 관리자 대시보드**

AI 식물 추천 O2O 플랫폼에서 PortOne V2 기반 PG 결제 시스템을 구축한 프로젝트입니다. 본 프로젝트의 "교육 신청 → PG 결제 → 상태 관리" 요건과 직접적으로 연결됩니다.

사용자가 상품을 선택하면 PortOne V2 결제창이 호출되고, 결제 완료 시 Webhook으로 서버에 통보되어 주문 상태가 자동 업데이트됩니다. 결제 실패, 환불, 부분 취소 등 예외 상황에 대한 처리 로직도 구현했습니다. 판매자 관리 대시보드에서는 주문 상태 트래킹(대기→결제완료→준비중→발송→완료), 매출 분석, 정산 관리를 통합 처리합니다.

특히 국세청 API를 연동한 사업자 인증, 은행 계좌 검증 등 외부 시스템 연동 경험이 있어, 본 프로젝트의 외부 LMS 연동에도 안정적으로 대응할 수 있습니다. React + Express + PostgreSQL + Drizzle ORM + PortOne V2 + Google Maps API 기반.

**3. Studio HaeTae (haetae.studio) - 반응형 웹 + 인증 + PDF 생성**

자체 에이전시 웹사이트로 Next.js App Router + Supabase 기반 반응형 웹사이트를 직접 설계·개발·운영 중입니다. 본 프로젝트와 동일한 기술 스택(Next.js + Supabase + Vercel)으로, 상담 접수 폼, 로그인/회원가입/비밀번호 찾기, 클라이언트 포털(대시보드)까지 구현했습니다.

특히 n8n 자동화 파이프라인에서 견적서 PDF 자동 생성 시스템을 구축한 경험이 있어, 본 프로젝트의 수료증/자격증 PDF 자동 생성에 동일한 패턴(HTML 템플릿 + Puppeteer → PDF 변환 + Supabase Storage 저장 + 발급 이력 DB 관리)을 적용할 수 있습니다. Vercel 배포로 글로벌 CDN, 자동 SSL, 무중단 배포를 확보하고 있으며, Core Web Vitals 최적화 경험도 보유하고 있습니다.
