[제안 개요]

클라이언트 디자인 제공 기준, 4주 내 완료 가능한 3단계 개발 계획입니다.

1단계 - 핵심 뽑기 시스템 (450만원/2주)
회원가입/로그인, 확률 기반 랜덤 뽑기 로직(100% 당첨), 조작 방지 보안 로그, PG 결제 연동 및 포인트 충전 시스템을 구축합니다. 서버 사이드 랜덤 생성과 SHA-256 해시 로그로 투명성을 확보합니다.

2단계 - 글로벌 확장 + UX 강화 (300만원/1주)
한국어/일본어/영어 3개국어 지원, 당첨 이펙트 애니메이션, Firebase 실시간 푸시 알림을 적용합니다.

3단계 - 관리자 대시보드 (250만원/1주)
상품 세트/확률 설정, 매출 통계, 다국어 콘텐츠 관리, 로그 모니터링 기능을 포함합니다.
