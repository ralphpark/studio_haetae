# 프리모아 프로필 Q&A

## Q1. 어떤 프로젝트를 주로 진행하시나요?

기업 홈페이지, 랜딩페이지부터 인력 매칭 플랫폼, 관리자 대시보드, SaaS 시스템까지 다양한 규모의 웹 프로젝트를 진행합니다. 특히 사업 초기 단계의 MVP 구축에 강점이 있어, 핵심 기능만 빠르게 만들고 검증한 뒤 단계적으로 확장하는 방식을 선호합니다. 최근에는 Education LMS, Local Commerce 플랫폼, 데이터 시각화 대시보드 등을 수행했습니다.

## Q2. 다른 개발자와 차별화되는 강점은 무엇인가요?

12년간 IT 대기업에서 CTO로 근무한 경험이 있어, 단순 코딩이 아닌 사업 관점에서 프로젝트를 바라봅니다. "이 기능이 정말 필요한가?", "사용자가 실제로 이렇게 쓸까?"를 먼저 고민하고, 불필요한 개발 비용을 줄이는 방향으로 제안드립니다. 기획부터 디자인, 프론트엔드, 백엔드, 배포까지 혼자 처리할 수 있어 커뮤니케이션 비용도 최소화됩니다.

## Q3. 프로젝트 진행 방식은 어떻게 되나요?

요구사항 정의 → 디자인 시안 → 개발 → 중간 검수 → 최종 납품 순서로 진행합니다. 대금은 선금 30%, 중도금 40%, 잔금 30%로 분할하여 부담을 줄여드리고, 개발 중간에 반드시 시연을 통해 방향이 맞는지 확인합니다. 납품 후 1개월간 무상 버그 수정을 제공하며, 이후 월 단위 유지보수도 가능합니다.

## Q4. 견적은 어떤 기준으로 산정하시나요?

기능별 공수(MD, Man-Day)를 기준으로 투명하게 산정합니다. 예를 들어 회원가입 3MD, 관리자 대시보드 10MD처럼 기능 단위로 공수를 산출하고, 이를 기반으로 견적서를 작성합니다. 예산이 제한적인 경우 MVP(핵심 기능)만 먼저 개발하고 이후 확장하는 단계별 접근도 제안드립니다.

## Q5. 프로젝트 완료 후 유지보수도 가능한가요?

네, 가능합니다. 납품 후 1개월간 개발 결함에 대해 무상으로 수정해 드리며, 이후에는 월 정기 유지보수 계약을 통해 기능 추가, 버그 수정, 서버 관리 등을 지원합니다. 코드와 서버 접근 권한은 모두 클라이언트에게 인수인계하므로, 유지보수를 다른 곳에 맡기셔도 문제없습니다.
